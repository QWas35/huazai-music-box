# 华仔音乐盒 Android APK 打包指南

## 📱 快速打包方案

由于 Android APK 打包需要 Linux 环境，提供以下几种方案：

---

## 方案一：GitHub Actions 自动打包 ⭐ 推荐

**最简单的方式，完全免费，无需本地环境！**

### 步骤：

1. **创建 GitHub 仓库**
   - 登录 GitHub.com
   - 创建新仓库，如 `huazai-music-box`

2. **上传代码**
   ```bash
   cd C:\Users\lenovo\.qclaw\workspace\华仔音乐盒_android
   git init
   git add .
   git commit -m "初始化华仔音乐盒 Android 版本"
   git branch -M main
   git remote add origin https://github.com/你的用户名/huazai-music-box.git
   git push -u origin main
   ```

3. **等待自动构建**
   - GitHub Actions 会自动开始构建（约 15-20 分钟）
   - 点击仓库的 "Actions" 标签查看进度

4. **下载 APK**
   - 构建完成后，在 Actions -> Artifacts 中下载 `huazaimusicbox-apk`
   - 或在 Releases 页面下载发布的 APK

## 方案二：本地 Docker 打包

**需要 Docker Desktop 正在运行**

```powershell
# 1. 启动 Docker Desktop

# 2. 进入项目目录
cd C:\Users\lenovo\.qclaw\workspace\华仔音乐盒_android

# 3. 使用 Docker 打包
docker run --rm -v "${PWD}:/app" -w /app kivy/buildozer android debug
```

## 方案三：WSL2 本地打包

**需要安装 WSL2 Ubuntu**

```powershell
# 1. 安装 WSL2 Ubuntu
wsl --install -d Ubuntu-22.04

# 2. 进入 WSL
wsl -d Ubuntu-22.04

# 3. 在 WSL 中执行
sudo apt update
sudo apt install -y git zip unzip openjdk-17-jdk autoconf libtool pkg-config zlib1g-dev libncurses5-dev cmake libffi-dev libssl-dev

pip3 install buildozer cython

# 复制项目到 WSL
cp -r /mnt/c/Users/lenovo/.qclaw/workspace/华仔音乐盒_android ~/
cd ~/华仔音乐盒_android

# 打包
buildozer android debug
```

## ⚠️ 常见问题

### 1. 打包失败：内存不足
- GitHub Actions 默认 7GB 内存，足够使用
- 本地打包建议至少 8GB 内存

### 2. 打包时间过长
- 首次打包需要下载 SDK/NDK，约 15-30 分钟
- 后续增量打包约 2-5 分钟

### 3. APK 安装失败
- 确保 Android 版本 >= 5.0
- 开启"允许安装未知来源应用"
